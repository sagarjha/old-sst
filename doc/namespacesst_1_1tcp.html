Exchanges data with a remote node using a TCP socket.

This will first send data to a remote node using the socket, then wait for the remote node to send the same amount of data back.

Parameters
sockThe file descriptor of the socket to use for the remote connection
xfer_sizeThe number of bytes of data to exchange.
local_dataA buffer containing the data to send to the remote node; must be at least xfer_size bytes large.
remote_dataA buffer into which the remote node's response data will be written; must be at least xfer_size bytes large.
Returns
The return code of the read() call that received data from the remote node.
